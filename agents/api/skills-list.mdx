Catalog of installable skills from glide.co/skills. Each skill advertises its runtime compatibility and whether its upstream dependencies are live in this environment.

## Metadata

| Field | Value |
| ----- | ----- |
| Name | `skills.list` |
| Category | `read` |
| Required scope | `agents:read` |
| Idempotency key required | no |

## Annotations

| Annotation | Value |
| ---------- | ----- |
| Title | Browse Skills |
| Read-only | yes |
| Destructive | no |
| Idempotent | yes |
| Open-world | no |
| Requires human approval | no |
